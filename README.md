# Inventory-Management-System
Smart inventory management system
--------------------------

1) create your own branch in the web

2)create data base in phpmyadmin "inventory_system".

3)open the cloned folder in vscode editor 

4) switch to your branch in local repo

5)create database in phpmyadmin "inventory_system".



6) import "inventory_system.sql" file  inside the DATABASE FILE folder (phpmyadmin has option to import sql files)

   
7)there is a  folder named Include, and inside it you can see config.php , adjest the variables relevent to your localhost configurations
